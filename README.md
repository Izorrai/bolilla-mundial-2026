# Bolilla Mundial 2026

App fantasy para una porra (bolilla) del Mundial 2026.

## Reglas

- Presupuesto: **1.050 puntos** por persona, 5€ por participación.
- Hay que elegir **20 selecciones**: 3 del Grupo A, 3 del B, 4 del C, 4 del D, 2 del E y 4 del F.
- Solo cuentan goles y resultados de los **90 minutos** (sin prórroga).

### Puntuación por selección

| Concepto | Puntos |
|---|---|
| Gol marcado | 0,5 |
| Victoria | 3 |
| Empate | 1 |
| Derrota | 0 |
| Pasar a dieciseisavos | 5 |
| Pasar a octavos | 7 |
| Pasar a cuartos | 10 |
| Pasar a semifinales | 15 |
| Pasar a final | 25 |
| Campeón | 50 |

### Extras

- Acertar Pichichi: 30 pts
- Acertar MVP del torneo: 30 pts
- Acertar Mejor portero: 30 pts

## Cómo se usa

Abrir [index.html](index.html) en el navegador (o usar la versión online en https://izorrai.github.io/bolilla-mundial-2026/). Marcar equipos, rellenar datos personales, imprimir o exportar a PDF para entregar la hoja con los 5€.

Funciona offline. Guarda automáticamente la selección en el localStorage del navegador.
